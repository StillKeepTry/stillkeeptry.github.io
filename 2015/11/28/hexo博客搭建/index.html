安装 Hexo 完成后,请执行下列命令,Hexo 将会在指定文件夹中新建所需要的文件.

新建完成后,文件夹目录如下:

目录文件说明

_config.yml

网站的 配置 信息,您可以在此配置大部分的参数.

package.json

应用程序的信息. EJS, Stylus 和 Markdown renderer 已默认安装,您可以自由移除.

scaffolds

模板 文件夹. hexo根据scaffold来建立文件.

source

资源文件夹是存放用户资源的地方
插入图片可以采用![](/images/avatar.jpg)

themes

主题 文件夹
