机器学习 -- 逻辑回归

發表於 2015-12-28   |   分類於 机器学习   |  

机器学习学习记录

定义

逻辑回归实际上是一个分类模型,但同时也被称为回归.
逻辑回归是一个二分类模型
逻辑回归是线性分类模型

模型描述

假设:

$P(y = 1 | x; k) = h_k(x)=g({k}^T x) = \frac{1}{1+e^{-k^T x}}$

$P(y = 0 | x; k) = 1 - h_k(x)$

联合形式:

$P(y | x; k) = {(h_k(x)) ^ {y}} {(1 - h_k (x))^{1-y}}$

参数: $k$

最大似然估计

条件似然

$L(k) = P(\overrightarrow{y}| X ; k)$

$L(k) = \prod_{i=1}^m p(y^{(i)} | x^{(i)} ; k)$

$L(k) = \prod_{i=1}^m (h_k(x^{(i)})^{y^{(i)}}) (1 - h_k(x^{(i)})^{1 - y^{(i)}})$

log 似然(交叉熵代价函数)

$\gamma (k) = \log L(k)$

$\gamma (k) = \sum_{i=1}^{m} y^{(i)}\log(h_k(x^{(i)}) + (1- y^{(i)})\log(1 - h_k(x^{(i)}))$

$argmax \sum_{i=1}^{m} y^{(i)}\log(h_k(x^{(i)}) + (1- y^{(i)})\log(1 - h_k(x^{(i)}))$

常用的优化方法

  • 梯度下降(Gradient Descent)
  • 随机梯度下降(Stochastic Gradient Descent)
  • 牛顿法(Newton Method)
  • 拟牛顿法(Quasi-Newton Method)
  • 共轭梯度(Conjugate Gradient)

梯度下降

已经知道我们的目标是要
$argmax \sum_{i=1}^{m} y^{(i)}\log(h_k(x^{(i)}) + (1- y^{(i)})\log(1 - h_k(x^{(i)}))$
根据梯度下降的定义,那么我们更新的过程应当如下:

目标函数的一阶导数如下:

$\frac{d(L(k))}{d k} = \sum_{i=1}^m (y^i - h_k (x^i)) x^i $

抱歉,latex 显示问题很大

$K = K + \alpha * \sum_{i=1}^m (y^i - h_k(x^i)) x^i $

随机梯度下降

和上面很想,只是取单样本,而不是整体.
最优化函数为
$K = K + \alpha * (y^i - h_k(x^i)) x^i $

牛顿法

牛顿法相比较于上面的最优化算法.牛顿法的可以很快收敛,应该说是非常快了.牛顿法采用了泰拉展开的近似解,而且牛顿法的精度也非常好,在没有归一化的基础,依然可以做好非常好的基础.

对于问题 $argmin f(x)$ 可以等价于 $\bigtriangledown f(x) = 0$

所以,更新过程就可以变成:

$x^(k+1) = x^k - {\bigtriangledown}^2 f(x^k) ^ {-1} {\bigtriangledown}f(x^k)$

其中 $ {\bigtriangledown}^2 f(x^k) $ 是海森矩阵
